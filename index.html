Sauce Chef — Physical Controller Game

A hotdog saucing arcade game controlled by a custom-built physical ketchup bottle. Three coloured buttons on the front select the sauce type; a pressure pad on the reverse triggers the dispense. Hotdogs travel across the screen on a conveyor — an invisible trigger zone at the left edge checks whether the correct sauce has been applied, awards points if correct, and immediately ends the run if wrong or missed. Game objects are destroyed after passing the check to prevent off-screen clutter and maintain performance. Built entirely solo — hardware wired from scratch, all game logic written in C#.
